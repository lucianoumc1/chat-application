## Getting Started 
***
Clone respository:
```
git clone https://github.com/lucianoumc1/chat-application.git
```

Install the dependencies:
```
npm install
// or
yarn
```

Run in dev mode.
```
$ npm start
// or
$ yarn dev
```
#### Built With
+ CSS Vanilla
+ JavaScript
+ [CreateReactApp](https://create-react-app.dev/)
+ [FireBase](https://firebase.google.com/)
